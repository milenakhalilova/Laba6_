Мой родной город - Уфа
Главная

Уфа - город-миллионник в России, столица Республики Башкортостан, административный центр Уфимского района, в состав которого не входит.
Город республиканского значения, образует муниципальное образование города Уфа со статусом городского округа.
Центр Уфимской агломерации.
Входит в число различных экономических, культурных и научных центров Российской Федерации.
